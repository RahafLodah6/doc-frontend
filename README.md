# Official Letters Assistant – Frontend

This is the frontend part of the **Official Letters Assistant** project for **Labayh** platform.  
It is built using **Next.js 14**, **TypeScript**, and **React**.

---

## Project Structure

- **`src/app/`** – Contains all pages, components, and global styling.
- **`src/components/`** – Reusable UI components such as `AgentBox`, `Preview`, and `PoliciesBox`.
- **`public/`** – Contains static assets such as the Labayh logo and icons.
- **`package.json`** – Lists all project dependencies.

---
